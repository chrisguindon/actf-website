ACTF 0.8 M2 - New and Noteworthy

WCAG 2.0 support (experimental)

We started to implement WCAG 2.0 check functions. Currently, however, this function is experimental. We'll continue to improve quality of check functions in next release.

Performance improvement

In some cases, it took a long time to finish usability visualization. We analyzed bottlenecks, and refined several items related to HTML Parser implementation and usage of XPath evaluation. (see bug 281318)

Bug fixes

We focused to find and fix bugs. Several bugs were fixed in this release.
